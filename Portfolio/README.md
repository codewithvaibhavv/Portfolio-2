# Vaibhav Sharma - Portfolio Website

A modern, dynamic, and responsive portfolio website showcasing my skills, experience, and projects as a Jr. .NET Developer.

## 🌟 Features

- **Modern Design**: Clean and professional UI with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects, scroll animations, and dynamic content
- **Smooth Navigation**: Sticky navbar with smooth scrolling to sections
- **Contact Form**: Integrated contact form with email functionality
- **Social Links**: Direct links to LinkedIn, GitHub, and email
- **Performance Optimized**: Fast loading and smooth animations

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript**: Interactive features and dynamic content
- **Font Awesome**: Icon library for beautiful icons

## 📂 Project Structure

```
portfolio/
│
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md          # Project documentation
```

## 🚀 Getting Started

1. **Clone or download** this repository
2. **Open** `index.html` in your web browser
3. **Customize** the content with your own information

### Local Development

Simply open the `index.html` file in any modern web browser. No build process or server required!

## 🎨 Customization

### Colors
The color scheme is defined using CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    --dark-bg: #0f172a;
    --darker-bg: #020617;
    --light-text: #f1f5f9;
    --gray-text: #94a3b8;
    --card-bg: #1e293b;
}
```

### Content
Update the content in `index.html` to reflect your own:
- Personal information
- Work experience
- Projects
- Skills
- Education
- Contact details

## 📱 Sections

1. **Hero Section**: Eye-catching introduction with name and title
2. **About**: Professional summary and education background
3. **Experience**: Work history with detailed descriptions
4. **Projects**: Showcase of academic and personal projects
5. **Skills**: Technical skills, tools, certifications, and soft skills
6. **Contact**: Contact form and social media links

## ✨ Key Features

- **Typing Effect**: Animated typing effect for the job title
- **Scroll Animations**: Elements fade in as you scroll
- **Parallax Effects**: Subtle mouse-tracking effects
- **Progress Bar**: Visual scroll progress indicator
- **Back to Top Button**: Quick navigation to the top
- **Mobile Menu**: Responsive hamburger menu for mobile devices
- **Active Navigation**: Highlights current section in navbar
